### 2. ⭐ **Система рейтинга и отзывов** ✅

**Функционал:**

#### Для пользователей:
1. После принятия жалобы получают уведомление о возможности оценки
2. Команда `/rate` — оценить последнюю выполненную жалобу
3. Выбрать рейтинг 1-5 звезд
4. Написать отзыв (опционально, можно /skip)

**Пример диалога:**
```
Пользователь: /rate

Бот: ⭐ Оценка жалобы №42
     Оцените качество выполненной работы от 1 до 5:
     1 - Очень плохо
     2 - Плохо
     3 - Удовлетворительно
     4 - Хорошо
     5 - Отлично
     
     Отправьте цифру от 1 до 5:

Пользователь: 5

Бот: ⭐⭐⭐⭐⭐
     
     Спасибо за оценку! Теперь напишите отзыв о выполненной работе.
     
     Или отправьте /skip чтобы пропустить отзыв.

Пользователь: Отличная работа! Все починили быстро.

Бот: ✅ Спасибо за оценку и отзыв!
     
     ⭐⭐⭐⭐⭐
     
     Ваше мнение поможет улучшить качество обслуживания.
```

#### В админ-панели:
**Новая страница `/ratings`** с:
- 🏆 Рейтинг работников (с медалями 🥇🥈🥉)
- Средний рейтинг каждого работника
- Количество принятых жалоб
- Количество полученных оценок
- 💬 Последние 10 отзывов

**Файлы:**
- `bot/database.py` — добавлены поля rating, review, rated_at
- `bot/states.py` — добавлен RatingForm
- `bot/handlers/user.py` — handlers для рейтинга
- `web/main.py` — route /ratings
- `web/templates/ratings.html` — страница рейтингов
- `web/templates/dashboard.html` — добавлена ссылка на рейтинги

## 🧪 Как протестировать:

### 1. Пересобрать Docker:
```bash
sudo docker compose build
sudo docker compose up -d
```

### 2. Проверить работу:

**Сценарий A: Рейтинг**
1. Пользователь: Подать жалобу `/complaint`
2. Работник: Принять жалобу (получить инфо о работнике)
3. Пользователь: `/rate` → выбрать оценку → написать отзыв
4. Админ: Открыть http://localhost:8000/ratings → увидеть рейтинг

**Сценарий B: Уведомления**
1. Подать жалобу → получить 2 сообщения (подтверждение + статус)
2. Работник принимает → получить детальное уведомление с ФИО, должностью, участком
3. Работник отклоняет → получить причину

### 3. Проверить логи:
```bash
sudo docker compose logs bot | grep "⭐"
```
